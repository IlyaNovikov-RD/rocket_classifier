------------------------------------------------------------------------

value for every prediction, grounded in cooperative game theory.
TreeExplainer computes exact (not approximate) Shapley values for tree-
based models in polynomial time. The reported 'Mean |SHAP|' is averaged
across all test trajectories and all three rocket classes — it
quantifies how much each physics feature moves the model's output, on
average.
